Note:
Some pins have default settings:

P0.00 and P0.01 are used for the 32.768 kHz crystal and are not available on the connectors. For more information, see 32.768 kHz crystal.
P0.05, P0.06, P0.07, and P0.08 are used by the UART connected to the interface MCU. For more information, see Virtual COM port.
P0.09 and P0.10 are by default used by NFC1 and NFC2. For more information, see NFC antenna interface.
P0.11–P0.16 and P0.24–P0.25 are by default connected to the buttons and LEDs. For more information, see Buttons and LEDs.
0.17 and 0.19–0.23 are by default connected to the external memory. For more information, see External memory.
